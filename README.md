# Laluan Career Guidance Platform

## Project Overview

Laluan is an AI-powered career guidance platform designed to help university students explore career opportunities, discover internships, and receive personalized career recommendations.

The platform aims to reduce uncertainty in career planning by providing students with accessible guidance, relevant opportunities, and AI-assisted support.

## Current Development Stage

This project is currently in the proposal and planning phase as part of a Final Year Project at Taylor's University.

## Planned Features

- AI Career Guidance Chatbot
- Personalized Career Recommendations
- Internship and Opportunity Discovery
- Student Profile Management
- Career Development Resources
- User Feedback and Progress Tracking
