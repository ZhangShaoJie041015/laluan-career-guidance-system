# Laluan Career Guidance Platform

## Project Overview

Laluan is an AI-powered career guidance platform designed to help university students explore career opportunities, discover internships, and receive personalized career recommendations.

The platform leverages Artificial Intelligence and Large Language Models (LLMs) to analyze student profiles, identify skill gaps, and generate personalized career development roadmaps. It aims to improve career readiness by providing adaptive guidance, learning recommendations, and career planning support.

## Current Development Stage

This project is currently in the proposal and planning phase as part of a Final Year Project at Taylor's University.

## Planned Features

- AI Career Guidance Chatbot
- Personalized Career Recommendations
- Career Roadmap Generator
- Skill Gap Visualizer
- Guidance Loop Dashboard
- Resume Improvement Tool
- Interview Practice Feature
- Learning Resource Suggestions
- Campus Career Calendar
- Verified Skills Passport
- Progress and Milestone Tracker
